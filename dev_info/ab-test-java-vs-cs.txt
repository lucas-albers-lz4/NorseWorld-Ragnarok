NorseWorld Ragnarok — Java v0.11 vs C# A/B test (Oldman / DemonBane quest)
============================================================================

Goal
----
Run the same NPC quest in both builds and record whether scripting (conditions,
actions, item transfer) behaves correctly.

Hypothesis
----------
Java: javax.script runs dialog JS (NPC.hasItem, NPC.isFieldCleared, transferItem).
C# (before fix): ExecuteScript() stubbed → conditions/actions do not run.
C# (after fix):  Jint + ScriptCreature restores dialog scripting on Mono.

- wine, curl, unzip          (Java launcher)
- mono-complete              (C# launcher; includes xbuild)

    ../BSLib
    ../ZRLib
